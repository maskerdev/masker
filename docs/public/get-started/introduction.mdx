**Masker is a real-time PHI redaction proxy for voice AI agents.** Point your Vapi or Retell assistant's Custom LLM URL at Masker, and PHI (names, dates of birth, SSNs, MRNs, account numbers) is tokenized before it ever reaches OpenAI, Anthropic, or your self-hosted LLM — then rehydrated in the response so callers hear a normal answer. Self-hosted as a single Rust binary or run on our managed beta. Every session ships with an Ed25519-signed HIPAA compliance report.

<Note>
**Status: private beta.** The portal at [masker-voice.fly.dev](https://masker-voice.fly.dev) and the live demo at [try.masker.dev](https://try.masker.dev) are functional today. Production beta launches **May 30, 2026**. To request access, see [Beta access](/get-started/beta-access).
</Note>

## Quick start: one URL change for Vapi

The fastest way to make a Vapi voice agent HIPAA‑safer is to swap its **Custom LLM URL** for a Masker proxy URL. No SDK install. No code change in your app.

```bash
# 1. Sign in to the Masker portal and create an agent
open https://masker-voice.fly.dev/portal/login

# 2. Copy the per-agent proxy URL (shown in the wizard)
#    Format: https://masker-voice.fly.dev/proxy/{agent_id}/v1/chat/completions

# 3. In Vapi → Assistant → Model:
#    Provider:        Custom LLM
#    Custom LLM URL:  <paste the Masker proxy URL>
#    Model:           gpt-4o-mini   (or your choice)

# 4. Place a test call. PHI is tokenized in transit:
#    "Hi, I'm Sarah Johnson, DOB March 14 1982"
#    → LLM sees: "Hi, I'm MSKV1.person_name.K_HEALTHCARE.a3f9..., DOB MSKV1.dob..."
```

That's the whole integration. The full walkthrough lives in the [Vapi integration tutorial](/integrations/vapi). For Retell, see [Retell + Masker](/integrations/retell).

## Frequently asked questions

<AccordionGroup>
  <Accordion title="What is Masker.dev?">
    Masker is a self-hosted HIPAA-grade privacy layer for voice AI agents. It sits between your voice platform (Vapi, Retell, Bolna) and your LLM, detects and tokenizes PHI before any text reaches the model, and rehydrates the original values in the LLM's response.
  </Accordion>
  <Accordion title="How do I make my Vapi agent HIPAA compliant with Masker?">
    Connect a Vapi agent in the Masker portal, copy the per-agent proxy URL, and paste it into Vapi as the Custom LLM URL. Every chat completion now flows through Masker, which tokenizes PHI before forwarding to OpenAI. Full walkthrough: [Vapi integration](/integrations/vapi).
  </Accordion>
  <Accordion title="Does Masker work with Retell AI?">
    Yes. Retell supports Custom LLM endpoints; the integration is the same one-URL change. See [Retell integration](/integrations/retell).
  </Accordion>
  <Accordion title="Can I self-host Masker in my VPC?">
    Yes. Masker ships as a single Rust binary and a Docker image. Run it on AWS, GCP, on-prem, or air-gapped. See [Self-hosted deployment](/guides/self-hosted).
  </Accordion>
  <Accordion title="What HIPAA Safe Harbor identifiers does Masker cover?">
    9 of the 18 Safe Harbor categories at full coverage today; 3 partial; 5 in progress (target: May 30, 2026). Full matrix: [HIPAA Safe Harbor coverage](/compliance/hipaa-safe-harbor).
  </Accordion>
</AccordionGroup>
