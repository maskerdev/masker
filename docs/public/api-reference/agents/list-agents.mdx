## Endpoint

```
GET /api/v1/agents
```

## Authentication

Requires `masker_session`.

## Query parameters

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| `limit` | int | `25` | Max results, up to `100` |
| `cursor` | string | (none) | Pagination cursor from a previous response |
| `name_contains` | string | (none) | Case-insensitive substring filter |

## Request

```bash
curl -H "Cookie: masker_session=ey..." \
  "https://masker-voice.fly.dev/api/v1/agents?limit=50"
```
