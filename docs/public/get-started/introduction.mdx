Masker is a HIPAA-grade privacy layer for voice AI agents. It sits between your voice platform (Vapi today; Bolna and Retell in beta) and your LLM, detects and redacts PHI and PII before any audio or text reaches the model, and rehydrates the original values on the way back so the caller's experience is unchanged.

<Note>
**Status: private beta.** The portal at [masker-voice.fly.dev](https://masker-voice.fly.dev) and the live demo at [try.masker.dev](https://try.masker.dev) are functional today. Production beta launches **May 30, 2026**. To request access, see [Beta access](/get-started/beta-access).
</Note>

## Honest scope

Masker is built and operated by a single founder (Navendra Jha, ex-Amazon Privacy Bar Raiser) with a small early-access cohort. These docs reflect what the product **actually does today** — not what it will do in v2. Where a feature is on the roadmap, it's marked clearly.

| Feature | Status |
| --- | --- |
| Vapi integration via per-agent proxy URL | ✅ Live |
| GitHub OAuth sign-in | ✅ Live |
| Real-time PHI/PII detection (regex + Gemma-4 NER) | ✅ Live |
| HMAC-SHA256 deterministic tokens · AES-256-GCM-SIV reversible tokens | ✅ Live |
| HIPAA Safe Harbor coverage (9 of 18 categories fully) | 🟡 Partial — full coverage by May 30, 2026 |
| Per-session signed compliance reports | ✅ Live |
| Voice pipeline (STT → mask → LLM → TTS) | ✅ Live |
| Bolna and Retell platform integrations | 🚧 In beta |
| Self-hosted Rust container in customer VPC | 🚧 Available on request |
| Multi-tenant teams + RBAC | 🚧 v2 |
