# ElevenLabs integration

1. In the ElevenLabs Conversational AI dashboard, open your agent settings.
2. Under **LLM**, choose **Custom LLM**.
3. Set the **Server URL** to your masker URL:
   `https://demo.masker.dev/agents/elevenlabs-public-demo/s/<your-session>/v1`
4. Leave the API key blank — masker's public demo is unauthenticated and pins the session via the URL path.
5. Save and start a conversation.
