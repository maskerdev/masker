## Endpoint

```
POST /api/v1/agents
```

## Authentication

Requires `masker_session`.

## Request body

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `name` | string | yes | Display name. Must be unique per account. 1-64 chars. |
| `upstream` | string | yes | One of `openai:gpt-4o-mini`, `openai:gpt-4o`, `stub` |
| `tokenization` | enum | no | `vault-deterministic` (default) or `reversible-aead` |
| `policy_name` | string | no | Name of an existing policy. Defaults to `healthcare-default`. |
| `policy_yaml` | string | no | Inline policy YAML. Mutually exclusive with `policy_name`. |

## Example

```bash
curl -X POST -H "Cookie: masker_session=ey..." \
  -H "Content-Type: application/json" \
  https://masker-voice.fly.dev/api/v1/agents \
  -d '{
    "name": "appointment-bot-prod",
    "upstream": "openai:gpt-4o-mini",
    "tokenization": "vault-deterministic"
  }'
```

## Errors

| Status | Code | Description |
| --- | --- | --- |
| `401` | `unauthenticated` | Missing session |
| `422` | `validation_failed` | Body invalid; see `details` |
| `409` | `name_taken` | An agent with this name already exists in the account |
| `422` | `unknown_upstream` | `upstream` doesn't match a configured provider |
| `422` | `unknown_policy` | `policy_name` doesn't exist |
| `422` | `policy_yaml_invalid` | Inline `policy_yaml` failed validation |
