---
title: "List sessions"
api: "GET /api/v1/sessions"
description: "List sessions across all agents in the account"
---

## Endpoint

```
GET /api/v1/sessions
```

## Authentication

Requires `masker_session`.

## Query parameters

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| `limit` | int | `25` | Up to `100` |
| `cursor` | string | (none) | Pagination cursor |
| `since` | string | `24h` | Duration or ISO timestamp |
| `agent_id` | string | (any) | Filter by agent |
| `status` | enum | (any) | `ok`, `error`, `partial` |
| `min_redactions` | int | `0` | Minimum redaction count |

## Request

```bash
curl -H "Cookie: masker_session=ey..." \
  "https://masker-voice.fly.dev/api/v1/sessions?since=24h&limit=50"
```

## Errors

| Status | Code | Description |
| --- | --- | --- |
| `401` | `unauthenticated` | Missing session |
